Reports any calls to System.out.println() with an exception as an argument. This is even worse than calling printStackTrace() on an exception, since it will be unclear where the exception originated. Calls to System.out.print(), System.err.println() and System.err.print() with an exception argument will also be reported. It is better to use logger to log exceptions instead.

E.g. instead of:


  try {
      foo();
  } catch (Exception e) {
      System.out.println(e);
  }

use the following code:


  try {
      foo();
  } catch (Exception e) {
      logger.warn(e); // logger call may be different
  }
